# 🎓 CampusPrep Hub

> **Your Complete AI-Powered Placement Preparation Platform**

CampusPrep Hub is a modern web application designed to help students prepare for campus placements through aptitude practice, coding challenges, technical MCQs, AI-powered resume analysis, mock interviews, and personalized learning. The platform provides an all-in-one solution to improve placement readiness with an intuitive and responsive user interface.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/mayurii35/campusprep-hub.git
cd campusprep-hub
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## 🔑 Environment Variables
Create a `.env` file inside the backend folder and configure the required environment variables such as:
* MongoDB Connection String
* Firebase Configuration
* Gemini API Key
* JWT Secret
